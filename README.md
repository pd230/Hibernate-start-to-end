# Java Web Sign-Up App

This is a Java web application using JSP, Servlets, Hibernate, and MySQL for user registration.

## Features
- User Registration
- Hibernate ORM
- JSP-based Frontend
- MySQL Database

## Tech Stack
- Java
- JSP & Servlets
- Hibernate
- MySQL
- Apache Tomcat

## How to Run
1. Clone the repo
2. Set up MySQL database and update `hibernate.cfg.xml`
3. Run on Tomcat server
